# Port - Model Portfolio Website

## Project Overview
모델(동생)을 위한 포트폴리오 웹사이트. 외부 캐스팅 디렉터/브랜드에게 전송 가능한 배포 버전.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS 4
- **Animation**: Framer Motion
- **Image**: next/image + Cloudinary
- **Font**: Playfair Display (EN) + Pretendard (KR)
- **Deploy**: Vercel

## Core Requirements
- 모바일에서 가볍게 작동 (3G에서도 3초 이내)
- 사진 관리 기능 (관리자 업로드/정렬/카테고리)
- 외부인이 쉽게 다양하게 열람 가능
- 사진 + 프로필 텍스트 균형 (6:4 비율)
- 배포 버전 (Vercel)

## Agent Team
- **카리나**: 디자인 시스템, UI/UX, 갤러리 레이아웃
- **프론트엔드 아키텍트**: Next.js 구조, 이미지 최적화, 성능
- **백엔드 아키텍트**: 사진 관리 CMS, 스토리지, 데이터 모델
- **데브옵스 아키텍트**: Vercel 배포, CDN, 도메인

## Design Direction
- Editorial Minimal (패션 매거진 스타일)
- 모노톤 + 골드 포인트
- 사진이 항상 주인공
- 풍부한 여백, 대담한 타이포그래피
